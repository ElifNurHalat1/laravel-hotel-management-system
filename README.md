# Laravel Hotel Management System

A Hotel Management System built with Laravel using modern authentication and a clean MVC architecture.

## Technologies
- Laravel
- Jetstream
- Livewire
- Blade
- Tailwind CSS
- MySQL
- Vite / npm

## Features
- Authentication (Login / Register)
- User Dashboard
- Profile Management
- Role-based structure (planned)
- Hotel & Room management (in progress)
